# Memory Directory

This directory stores long-term memory and context from conversations.

## Purpose

- Preserve important conversation context
- Store learned information and preferences
- Maintain continuity across conversation sessions

## File Structure

Files in this directory should be organized by:
- Date (YYYY-MM-DD format)
- Topic or project name
- Conversation ID

## Usage

Memory files are automatically managed by the conversation system. Manual editing is not recommended unless you understand the memory format.
