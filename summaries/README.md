# Summaries Directory

This directory contains automatically generated summaries of conversation sessions.

## Purpose

- Provide quick overview of conversation content
- Enable efficient review of past conversations
- Support search and retrieval of conversation topics

## File Naming Convention

Summaries follow this naming pattern:
- `YYYY-MM-DD_HH-MM-SS_topic.md` - Date, time, and topic
- `conversation-ID_summary.md` - Conversation ID based

## Auto-Generation

Summaries are automatically generated by the conversation system and include:
- Key topics discussed
- Action items identified
- Decisions made
- Important information extracted
