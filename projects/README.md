# Projects Directory

This directory organizes conversations by project or topic.

## Purpose

- Group related conversations together
- Maintain project-specific context
- Enable focused search within project boundaries

## Organization

Each project directory should contain:
- Project-specific conversation logs
- Relevant summaries and metadata
- Project context and notes

## Usage

Create subdirectories for each project:
```
projects/
├── project-name-1/
│   ├── conversations/
│   ├── summaries/
│   └── context.md
├── project-name-2/
│   ├── conversations/
│   ├── summaries/
│   └── context.md
└── ...
```
