# MBclaw-Lite

A lightweight claw machine simulation

## Description

MBclaw-Lite is a simple, browser-based claw machine simulation game. This project aims to provide a fun and interactive experience with realistic physics and controls.

## Features

- Simple and intuitive controls
- Realistic claw movement and grabbing mechanics
- Score tracking system
- Responsive design for various screen sizes

## Installation

To run this project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/mengbaiyoudianxian/MBclaw-Lite.git
   ```

2. Open the `index.html` file in your browser

Or use a local server:
```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve .
```

## How to Play

1. Use arrow keys or WASD to move the claw
2. Press Space or click the grab button to drop the claw
3. Try to grab prizes and score points!

## Development

This is a pure HTML/CSS/JavaScript project with no build process required.
