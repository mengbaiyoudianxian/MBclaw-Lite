# MBclaw-Lite

A lightweight system for managing, analyzing, and searching conversation data.

## Project Structure

```
MBclaw-Lite/
├── memory/          # Long-term memory storage for conversations
├── summaries/       # Auto-generated conversation summaries
├── projects/        # Project-specific conversation groupings
├── snapshots/       # Point-in-time conversation snapshots
├── prompts/         # Prompt templates and configurations
├── tools/           # Utility scripts and tools
└── docs/            # Documentation and guides
```

## Features

### Core Functionality
- **Save All Conversations**: Complete conversation history preservation
- **Auto Summary**: Automatic generation of conversation summaries
- **Auto Classification**: Intelligent categorization of conversations
- **Auto Keyword Extraction**: Automatic extraction of relevant keywords
- **Search Support**: Full-text search across all conversation data

### Directory Details

- **memory/**: Stores long-term memory and context from conversations
- **summaries/**: Contains auto-generated summaries of conversation sessions
- **projects/**: Organizes conversations by project or topic
- **snapshots/**: Saves point-in-time snapshots for reference
- **prompts/**: Houses prompt templates and configuration files
- **tools/**: Utility scripts for data processing and analysis
- **docs/**: Documentation, guides, and usage instructions

   ```bash
   git clone https://github.com/mengbaiyoudianxian/MBclaw-Lite.git
   ```

## Usage

This system is designed to work with conversation management tools. Each directory serves a specific purpose in the conversation lifecycle:

1. Conversations are captured and stored
2. Summaries are automatically generated
3. Conversations are classified and indexed
4. Keywords are extracted for searchability
5. All data becomes searchable and retrievable
